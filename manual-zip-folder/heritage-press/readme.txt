A comprehensive genealogy management system for WordPress.

== Description ==

Heritage Press is a powerful WordPress plugin designed for genealogy enthusiasts who want to manage and display their family history directly on their WordPress website. This plugin provides a complete solution for organizing, importing, and presenting genealogical data.

**Key Features:**

* **GEDCOM Import/Export** - Full support for GEDCOM 7.0 format
* **Individual & Family Management** - Complete person and family relationship tracking
* **Evidence Documentation** - Following Evidence Explained standards
* **Media Management** - Photos, documents, and other media attachments
* **Source Management** - Comprehensive source citation system
* **Advanced Search** - Find individuals, families, and records quickly
* **Responsive Design** - Mobile-friendly interface
* **WordPress Integration** - Seamless integration with WordPress themes

**Import Support:**

* GEDCOM files (versions 5.5 to 7.0)
* RootsMagic databases
* Family Tree Maker files
* CSV data import

**Display Features:**

* Individual profiles with photos and biographical information
* Family group sheets
* Pedigree charts
* Descendancy charts
* Timeline views
* Interactive family trees

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/heritage-press` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Heritage Press menu in your WordPress admin to configure the plugin.
4. Import your genealogical data using the Import tools.

== Frequently Asked Questions ==

= What file formats can I import? =

Heritage Press supports GEDCOM files (versions 5.5 to 7.0), RootsMagic databases, and CSV files.

= Is my data secure? =

Yes, all data is stored in your WordPress database with proper security measures. You can also export your data at any time.

= Can I display family trees on my website? =

Yes, Heritage Press includes shortcodes and widgets to display family trees, individual profiles, and other genealogical information on your website.

= Does it work with my theme? =

Heritage Press is designed to work with any properly coded WordPress theme. It uses standard WordPress styling practices.
